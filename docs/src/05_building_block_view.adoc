== Building Block View

=== Whitebox Overall System

_**Overview Diagram**_

image::Level1-Building-Blocks-Overview.png["Building Block View (Level 1)"]

Motivation::
La descomposición separa claramente interfaz, servicios y persistencia.
Esto facilita mantener el sistema, probar cada parte por separado y evolucionar
cada bloque sin afectar al resto.

Contained Building Blocks::
[cols="1,3" options="header"]
|===
| Name | Responsibility
| Webapp | Aplicación SPA. Gestiona pantallas, estado de juego en cliente y coordinación de llamadas a APIs.
| Users Service | Servicio de autenticación. Registra usuarios, valida credenciales y devuelve respuestas de login/registro.
| GameY Engine | Motor del juego Y. Aplica reglas, valida jugadas y ofrece endpoints de bot y movimiento.
| MongoDB | Persistencia común con dos colecciones: `users` (usuarios) y `games` (resultados de partidas).
|===

Important Interfaces::
Las interfaces clave se describen en el nivel 2.

=== Level 2

==== White Box Webapp

_**Diagram**_

image::Level2-WebApp.png["Building Block View (Level 2 - Webapp)"]

Responsibility::
Controla el flujo de usuario (inicio → menú → partida), renderiza el tablero
y coordina la comunicación con los servicios.

Contained Building Blocks::
[cols="1,3" options="header"]
|===
| Component | Responsibility
| App.tsx | Orquesta las vistas principales.
| RegisterForm.tsx | Login/registro y validaciones en cliente.
| Menu.tsx | Selección de modo, tamaño y variante.
| GameBoard.tsx | Render del tablero y control de turnos.
| GameyApi.ts | Cliente HTTP hacia GameY.
|===

==== White Box Users Service

_**Diagram**_

image::Level2-UsersService.png["Building Block View (Level 2 - Users Service)"]

Responsibility::
API de autenticación. Verifica entradas, genera hash de contraseñas y
persiste usuarios en MongoDB.

Contained Building Blocks::
[cols="1,3" options="header"]
|===
| Module | Responsibility
| users-service.js | Endpoints `/register`, `/login`, `/createuser` y lógica principal.
| openapi.yaml | Contrato de API.
|===

==== White Box GameY Engine

_**Diagram**_

image::Level2-GameY.png["Building Block View (Level 2 - GameY)"]

Responsibility::
Motor de reglas del juego. Recibe el estado (YEN), valida jugadas y
devuelve el nuevo estado, además del movimiento del bot.

Contained Building Blocks::
[cols="1,3" options="header"]
|===
| Module | Responsibility
| core/ | Estado del juego, reglas y validación.
| notation/ | Conversión a/desde YEN.
| bot/ | Estrategias de bot (random).
| bot_server/ | Endpoints HTTP.
| db/ | Persistencia de resultados finales.
|===
